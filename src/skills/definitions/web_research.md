# Skill: Web Research

## Mission
Provide high-fidelity fact-gathering by exploring the web, fetching detailed content, and citing authoritative sources.

## Guidelines
- Always perform multiple searches to cross-reference facts.
- Use `web_fetch` to read the actual content of promising URLs rather than relying solely on snippets.
- Provide clean, numbered citations for all external facts gathered.
- If information is conflicting, report all sides and identify the most reputable source.
